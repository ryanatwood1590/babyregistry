Google Sheets Connection

RSVPs submit to a Google Apps Script which writes to your Sheet
1
Create a Google Sheet with these headers in Row 1: Timestamp Name Attending
2
Extensions → Apps Script. Delete any existing code, paste the script below, and click Save.
3
Deploy → New deployment. Click the gear ⚙ next to Type and choose Web app. Set Execute as: Me and Who has access: Anyone. Click Deploy and authorize. Important: if you had an old deployment, you must create a NEW one — do not re-use the old URL.
4
Copy the Web app URL and paste it in the field below. It should look like https://script.google.com/macros/s/ABC.../exec. Then click Save.
function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Sheet1');
  var p = e.parameter;
  sheet.appendRow([
    p.timestamp || new Date(),
    p.name,
    p.attending
  ]);
  return ContentService
    .createTextOutput(JSON.stringify({result: 'success'}))
    .setMimeType(ContentService.MimeType.JSON);
}

🚀 Publish to GitHub

2. Go to your GitHub repo → click Add file → Upload files
3. Upload data.js alongside your index.html
4. Done — every visitor on every device now sees your edits ✓
